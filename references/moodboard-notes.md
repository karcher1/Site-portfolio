# Moodboard notes

Use these notes instead of copying existing games.

## Desired feeling

- professional RPG interface;
- dark fantasy dashboard;
- character profile screen;
- quest log / inventory / stats;
- structured, readable, slightly playful.

## Visual ingredients

- dark navy background;
- glowing panel borders;
- pixel corners;
- small iconography;
- stat bars;
- quest cards;
- inventory item cards;
- low-contrast map background.

## Avoid

- direct references to existing games;
- using screenshots from games;
- copyrighted characters;
- childish arcade colors;
- dense backgrounds behind text.
